This project contains code for an academicpages website.

### Overall Rules
* If a package is needed for installation, or if running a program fails because a missing dependency, prompt me before doing anything else. I may install the package manually.
* Don't silently catch exceptions. If you want to catch an exception, 1. ask me if its necessary and 2. for the desired behavior on exception.

### Loading files into memory

In the directory there may be a number of data files: 
* .jpg
* .csv
* .json
* .geojson
* .pdf

Generally speaking, you *do not* need to load any into context. If you feel that you do, seek approval first.
